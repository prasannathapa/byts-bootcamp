how this works

The test is the specification.

  • Each step, you pull that step's tests only. The solution code is never handed to you.
  • Open the tests and read them. They say exactly what your classes must do.
  • Write the classes. Run mvnw test until the step is green.
  • Tick mark step complete at the bottom of each step. The pill up top tracks you.

Stuck? The instructor is walking the same steps on the big screen. The step labels match.

Step 01start clean

Clone, then rewind to the scaffold.

# a fresh clone is the finished app; rewind to the empty scaffold
git clone <repo-url> && cd mini-instagram
git switch -c my-build step-01-scaffold
  • Confirm a JDK 25 or newer: java -version
  • You are now at the scaffold: build tooling and an empty Main, no solution code.
  • mvnw test passes the single scaffold test. You are ready.
