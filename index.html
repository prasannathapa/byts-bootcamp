Capstones

Each *-capstone/ folder is a small Java exercise. Clone or download this repo, read RunTests.java (the spec), fill in the TODOs, and run until every test is green. Requires a JDK 17 or newer; no build tools to install. 
git clone https://github.com/prasannathapa/byts-bootcamp
cd byts-bootcamp/oop-capstone      # or solid-capstone, patterns-capstone
./run.sh                           # Windows: run.cmd
Day 4's capstone is the full Mini Instagram project, in its own repo: github.com/prasannathapa/mini-insta.
